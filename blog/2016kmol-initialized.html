2016kmol initialized

Date 三 16 十二月 2015 Tags 啟動

https://github.com/chiamingyen/kmol2016 是一個 Windows 64 位元環境下的可攜程式系統.

製作步驟如下:

  1. 選擇安裝 Python 3.4.4, 原因為 Python 3.5 與 PyQt5 必須自行編譯, 為了簡化, 選擇 Python 3.4.4, 並且使用現成的 PyQt5-5.5.1-gpl-Py3.4-Qt5.5.1-x64.exe
  2. 安裝 Python 3.4.4 時, 因為要製作成可攜免安裝套件, 故意不安裝 pip, 隨後再透過 get-pip.py 以手動安裝.
  3. 安裝 pip 後, 再利用 Leo Editor 原始碼安裝, 同時利用 pip 安裝 pelican Markdown Flask
  4. 為了要使用 Tipue Search, 再安裝 beautifulsoup4
  5. 為了要執行協同產品開發的設計運算分析, 還要安裝 Jupyter
